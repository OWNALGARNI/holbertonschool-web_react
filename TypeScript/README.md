# TypeScript Project

## Description
This project is an introduction to TypeScript. It covers both basic and advanced concepts, starting from simple types and interfaces to more complex topics such as namespaces, declaration merging, and nominal typing.

Each task is organized in its own directory and focuses on a specific concept.

---

## Learning Objectives

At the end of this project, I learned how to:

- Use basic types in TypeScript
- Create and use interfaces
- Work with classes and functions
- Manipulate the DOM using TypeScript
- Use advanced types (union types, type guards)
- Work with namespaces
- Use declaration merging
- Import external libraries using ambient namespaces
- Apply nominal typing using branding

---

## Project Structure

- **task_0**: Basic interface and DOM manipulation  
- **task_1**: Interfaces and classes  
- **task_2**: Advanced types and type guards  
- **task_3**: Ambient namespaces and external libraries  
- **task_4**: Namespaces and declaration merging  
- **task_5**: Nominal typing and branding  

---

## Requirements

- All files are written in TypeScript
- Code is compiled using Webpack
- No TypeScript errors should be present
- Each file ends with a new line
